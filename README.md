# README — Python-Powered Cyber, AI & Cloud Shenanigans

## 🚀 About Me

Hi, I'm a Python enthusiast, I've spent way too many nights debugging indentation errors and calling it "fun."

I build tools across cyber operations, AI automation, and cloud workflows, because nothing says "living the dream" quite like writing scripts to fix problems caused by other scripts.

I explore:

- **Threat detection** — attackers don't sleep, and apparently neither do I
- **AI assistants** — my ongoing attempt to outsource my own brain
- **Cloud automation** — herding AWS and Azure like a pair of misbehaving cats
- **Everyday tools** — small scripts that make life about 0.3% easier

## 🧠 Mission Statement

I use Python to turn chaos into automation, confusion into clarity, and logs into something that doesn't make me cry.

My tools aim to be:

- **Lightweight** — unlike my caffeine consumption
- **Understandable** — unlike half of Stack Overflow
- **Operationally useful** — unlike most corporate dashboards
- **Easy to deploy** — unless YAML is involved, in which case all bets are off
